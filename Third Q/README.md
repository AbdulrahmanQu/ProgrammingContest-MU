السؤال الأول
اكتب برنامج يستقبل مجموعة من الاعداد  الصحيح N ثم يقوم بطباعة مجموع اول واخر خانة  من عدد N
0 < T <= 100
0 <= N <= 1000000
*المدخلات*
T عدد صحيح يساوي عدد حالات الاختبار
N الاعداد الصحيحة 
*المخرجات*
مجموع اول واخر خانة  من العدد N
